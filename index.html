Camina seguro con SafeWalk

SafeWalk es una aplicación móvil enfocada en la seguridad urbana. Permite activar alertas de emergencia, utilizar reconocimiento de voz, analizar zonas peligrosas y acceder a rutas más seguras en tiempo real.

Funciones principales

La aplicación integra herramientas modernas de seguridad urbana para ayudar a las personas durante trayectos nocturnos o situaciones de emergencia.

🚨

Modo SOS

Activa rápidamente una alerta de emergencia para notificar situaciones peligrosas y compartir información importante.

🎤

Reconocimiento de voz

El usuario puede hablar directamente con la aplicación mientras camina. La voz se transforma automáticamente en texto para detectar situaciones críticas.

📍

Ubicación en tiempo real

La app utiliza geolocalización para identificar calles, zonas y puntos de riesgo durante recorridos urbanos.

🗺

Rutas seguras

SafeWalk busca ayudar a las personas a evitar zonas peligrosas mediante rutas más seguras especialmente durante horarios nocturnos.

¿Cómo funciona?

SafeWalk está diseñado para reaccionar rápidamente ante posibles situaciones de peligro mientras el usuario se encuentra caminando por la ciudad.

1

Detectar

El usuario detecta una situación sospechosa o siente inseguridad mientras camina.

2

Activar

Se activa el modo de emergencia o el reconocimiento de voz mediante un botón rápido.

3

Analizar

La aplicación analiza el mensaje, la ubicación y posibles zonas de riesgo cercanas.

4

Responder

SafeWalk muestra alertas, información preventiva y futuras opciones de ayuda automática.

Sobre el proyecto

SafeWalk es un proyecto orientado a mejorar la seguridad urbana mediante el uso de tecnologías móviles, reconocimiento de voz, geolocalización y futuras integraciones con inteligencia artificial.

📱

Aplicación Android

Desarrollada en Kotlin utilizando Android Studio para dispositivos móviles Android.

🧠

IA y análisis futuro

El proyecto contempla futuras funciones basadas en inteligencia artificial y análisis de riesgo.

🌙

Seguridad nocturna

Pensado especialmente para personas que deben movilizarse de noche o por zonas desconocidas.
